# MANASCHMIEDE

**Hard requirement: great deck and card recommendations that synergize and are
fun to play.**

A Magic: The Gathering deck builder with AI-powered card suggestions. Built on
the **Specimen** design language — ink on ash, hairlines, sharp corners, zero
compromise.

## Stack

Convex · React 19 · TanStack Router · Tailwind 4 · Anthropic Claude

## Development

```sh
pnpm install
pnpm dev          # Vite dev server
pnpm dev:convex   # Convex local backend
```
