Greenfield project starting Phase 1 (April 2026). Tech: TanStack Start, Convex, Tailwind v4, pnpm.

**Why:** German MTG players lack a good deck builder with native German card display and balance analysis.

**How to apply:** Prioritize German localization throughout, use Scryfall API with lang=de, store only card IDs in Convex (not full card data), respect Scryfall 50ms rate limit.
